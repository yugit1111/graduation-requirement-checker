📘 卒業要件チェッカー（Graduation Requirement Checker）
※ 本リポジトリは学習目的のサンプルです。
※ 実際の卒業判定は各大学の公式情報を必ず確認してください。


学科・コースごとの卒業要件をもとに、
履修状況を入力することで 卒業可否を簡単に確認できる Web アプリ です。

🔍 概要

このアプリは、大学の卒業要件（必修・選択・単位数など）を入力し、
現在の履修状況が要件を満たしているかを視覚的に確認できるツールです。

必須科目の未履修を自動検出

単位数の過不足を即時表示

ブラウザ上で完結（インストール不要）

🧩 主な機能

✅ 科目の追加・削除

✅ 必修科目の一括チェック

✅ 単位数の自動集計

✅ 未履修科目の警告表示

✅ ローカル保存（リロードしてもデータ保持）

✅ 見やすいカード型UI

🖥️ 使用技術

HTML / CSS / TypeScript

ローカルストレージ（localStorage）

フレームワーク不使用（学習用）
